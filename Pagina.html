Objetivo del Curso

Meta General: Al finalizar el curso, los participantes habrán creado una página web funcional (o una parte específica de una web, como una landing page) utilizando HTML, CSS, y JavaScript básico.

Módulo 1: Fundamentos de Desarrollo Web

Subtema 1: Introducción al Desarrollo Web

Conceptos básicos: ¿Qué es una página web? ¿Cómo funciona el navegador?

Estructura básica de un archivo HTML.

Subtema 2: HTML y la Estructura de la Página

Etiquetas HTML esenciales: encabezados, párrafos, imágenes, enlaces, listas.

Creación de la estructura básica de la página web.

Subtema 3: CSS y el Diseño de la Página

Introducción a CSS: selectores, propiedades y valores.

Diseño básico de la página web: colores, fuentes, y espaciados.

Módulo 2: Funcionalidades y Estilo Avanzado

Subtema 1: Layout y Diseño Responsivo

Uso de cajas y contenedores: div, span, y otros.

Diseño responsivo con Flexbox y Grid.

Subtema 2: Introducción a JavaScript

Conceptos básicos: variables, funciones, eventos.

Interactividad básica: botones, formularios.

Subtema 3: Incorporación de Recursos Externos

Uso de iconos y tipografías externas.

Integración de imágenes, videos, y otros medios.

Módulo 3: Proyecto Final

Subtema 1: Planificación del Proyecto

Definición de la página web final: objetivos, público, funcionalidad.

Organización del contenido: estructura y diseño.

Subtema 2: Desarrollo del Proyecto
